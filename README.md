# COSC 4351 Backend

Express TypeScript backend to service cosc-4351-frontend

## Dependencies

- NodeJS 8.0.0
- TypeScript
- TSLint
- Express

## Usage

1. Install dependencies
```
npm install
```

2. Create `dist` folder in project root directory

3. Copy `server.js` into `dist`

4. Start server
```
npm start
```
